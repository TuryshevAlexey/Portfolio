AviaBot

Десктоп приложение для оптимизации работы специалиста по авиабилетам.

Топ

Приложение обладает следующим функционалом:

  • ведение отчётности по обработанным заявкам
  • генерация служебных записок
  • выполнение расчёта стоимости обмена билетов
  • предоставление справочной информации по выбранной авиакомпании
  • ведение реестра заявок на возврат, находящихся в обработке
  • отправка справочных и учебных материалов, а также контроль их прочтения

Другие проекты находятся в процессе разработки.
